# E-Cell VPKBIET Website

## Overview
This repository contains the source code for the official website of the Entrepreneurship Cell (E-Cell), VPKBIET.  
The website is designed to present the vision, initiatives, events, and organizational structure of the E-Cell while promoting entrepreneurship, innovation, and startup culture among students.

The platform serves as a centralized digital presence for outreach, event promotion, and student engagement.

---

## Objectives
- To provide a professional online identity for E-Cell VPKBIET  
- To showcase events, workshops, and entrepreneurial initiatives  
- To enable student participation through integrated application and registration links  
- To ensure accessibility across devices with a responsive design  

---

## Features
- Modern and professional **purple–gold theme**
- Responsive, mobile-friendly layout
- Smooth scroll-based animations
- Real event photo showcases
- Mobile navigation menu
- Google Form integration for registrations and applications
- Clean and structured UI for institutional use

---

## Technology Stack
- **HTML5** – Structure and content  
- **CSS3** – Styling, layout, and animations  
- **JavaScript (Vanilla)** – Interactivity and dynamic behavior
